# AI Assistant Feature Module

## Purpose
This feature module is structured to accommodate the AI Assistant chat prompts console and the system-level action prompt text parsing logic rules.

## Directory Structure
- `components/` - Assistant chat overlays and shortcut chips guides.
- `hooks/` - Text query submits and mock actions execution pipelines.
- `mock/` - Pre-loaded AI prompts answers and message history.
- `types/` - TS specifications.
