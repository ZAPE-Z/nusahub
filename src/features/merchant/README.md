# Merchant Feature Module

## Purpose
This feature module is structured to accommodate the UMKM Merchant Dashboard, store catalog managers (adding/editing products), and order fulfillment logs.

## Directory Structure
- `components/` - Quick stats panels and inventory checklist cards.
- `hooks/` - Product updates and order confirmations hooks.
- `mock/` - Sales logs and active orders arrays.
- `types/` - TS specifications.
