# Screen Inventory Index

## Executive Summary
This document catalog index lists every screen view required for the NusaHub MVP prototype. The inventory ensures complete screen coverage and provides a tracking catalog for designers and engineers.

## Main Analysis
Below are the 15 screen views required for the prototype:

### Module 1: Authentication
1. **SCR-01-Register**: Account registration form view.
2. **SCR-02-Login**: Sign-in portal screen view.

### Module 2: Core Shell & Feed
3. **SCR-03-HomeFeed**: Main feed displaying creator posts and catalog references.
4. **SCR-04-PostDetail**: Detailed thread view of a single creator post with comments.

### Module 3: Messaging Hub
5. **SCR-05-ChatsList**: Navigation tab listing active personal conversation logs.
6. **SCR-06-ChatRoom**: Active conversation messaging panel (with checkout trigger).

### Module 4: E-Commerce & Wallet
7. **SCR-07-Marketplace**: Marketplace list search portal.
8. **SCR-08-ProductDetail**: Catalog item detail page.
9. **SCR-09-CheckoutSheet**: Bottom overlay sheet validating orders.
10. **SCR-10-WalletCenter**: Wallet balances and transactions log list views.
11. **SCR-11-QRScan**: Mock camera scan interface view.

### Module 5: Merchant Dashboard
12. **SCR-12-InventoryList**: Seller inventory item log console.
13. **SCR-13-OrderFulfillment**: Active merchant orders manager.

### Module 6: Workspace & Profile
14. **SCR-14-WorkspaceHub**: Consolidated personal task, note, and calendar portal.
15. **SCR-15-UserProfile**: User profile settings page with system role switch triggers.

## Key Insights
- **Cohesive Screen Mapping**: Inventory codes (SCR-01 through SCR-15) establish traceability across user flows and technical code.
- **Prototype Scope**: The index covers only functional MVP screens, ensuring that no complex out-of-scope views are designed or coded.
